# Image Assets Guide

## Required Images

### Profile Images (`profile/`)
- **portrait.jpg** - Main profile photo
  - Recommended size: 800x1000px (4:5 aspect ratio)
  - Format: JPG or WebP
  - Professional headshot or full-body portrait

### Award Certificates (`awards/`)
- **unesco-cert.jpg** - UNESCO Award certificate
- **peace-cert.jpg** - International Peace Award certificate
- **tuc-cert.jpg** - TUC Humanitarian Award certificate
- **kano-cert.jpg** - Kano Contractors Association certificate
  - Recommended size: 1920x1080px or higher
  - Format: JPG or PNG
  - High resolution for zoom functionality

### Organization Logos (`logos/`)
- Organization and partner logos
  - Format: SVG (preferred) or PNG with transparency
  - Size: Variable, maintain aspect ratio

## Image Optimization Tips

1. **Compress images** before uploading (use tools like TinyPNG)
2. **Use WebP format** for better performance with JPG fallback
3. **Maintain aspect ratios** to prevent distortion
4. **Add alt text** for accessibility
5. **Use descriptive filenames** for SEO

## Placeholder Images

Until you add your own images, you can use placeholder services:
- https://via.placeholder.com/800x1000
- https://picsum.photos/800/1000

Replace all placeholder paths in HTML files with your actual image paths.
