## Image Optimization

### Before Uploading
1. Resize to recommended dimensions
2. Compress using [TinyPNG](https://tinypng.com)
3. Convert to WebP for better performance (optional)
4. Ensure file size < 500KB per image

### Tools
- **Resize**: Photoshop, GIMP, or online tools
- **Compress**: TinyPNG, ImageOptim
- **Convert**: CloudConvert, Squoosh

## Placeholder Images

Until you have real images, use:
- [Placeholder.com](https://placeholder.com)
- [Lorem Picsum](https://picsum.photos)
- [Unsplash Source](https://source.unsplash.com)

Example:
```html
<img src="https://via.placeholder.com/800x600" alt="Placeholder">
```

## Image Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `team-member-john-doe.jpg`
- ✅ `service-cloud-computing.jpg`
- ✅ `blog-post-cybersecurity-tips.jpg`
- ❌ `IMG_1234.jpg`
- ❌ `photo 1.jpg`

## Alt Text Guidelines

Always add descriptive alt text:
```html
<img src="images/logo.png" alt="ABU IT TECH LTD Logo">
<img src="images/team/member1.jpg" alt="John Doe, CEO of ABU IT TECH LTD">
<img src="images/services/it-support.jpg" alt="IT support technician helping client">
```
